# uScient

uScient is about user-owned knowledge and user-defined trust.

We are building toward a world where personal context, source authority, privacy, and information boundaries are inspectable, adjustable, portable, and governed by the user.

Privacy is part of this, but uScient is not merely privacy tooling. The deeper goal is manageable trust around personal knowledge.

uScient is agent-compatible, not agent-run. AI systems can help inspect, label, summarize, and route context, but authority remains with the user.

## Core Ideas

- **Your knowledge should remain yours.**
- **Trust should be defined by the user, not assumed by the platform.**
- **Context should be inspectable before it is shared.**
- **Sources should carry authority, provenance, and boundaries.**
- **Personal information control should be practical, not theoretical.**
- **AI systems should receive only the context they are meant to receive.**
- **Agents may operate within user-defined boundaries. They do not set them.**

## Current Focus

uScient is early-stage.

The current focus is practical, developer-readable work around:

- local-first context handling
- source-aware knowledge workflows
- inspectable trust boundaries
- AI-readable documentation
- personal information control
- open tools that earn trust through clarity and restraint

## Public Standard

uScient should be described with precision.

Not hype.  
Not vague platform language.  
Not empty privacy promises.  
Not agent-run autonomy.

The goal is to make trust manageable.
