# uScient GitHub Doctrine

This repository defines the public documentation doctrine for the uScient GitHub organization.

It is the language source for how uScient describes itself, what it means, what it does not claim, and how agents should maintain its public Markdown documents.

## What uScient Is

uScient is about user-owned knowledge and user-defined trust.

It exists to make personal context, source authority, privacy, and information boundaries inspectable, adjustable, portable, and governed by the user.

Privacy matters, but uScient is not merely "privacy tools." The larger goal is manageable trust around personal knowledge.

uScient is agent-compatible, not agent-run. AI systems and agents may assist users, but they do not define trust policy, own user knowledge, or become the authority over personal context.

## What This Repository Controls

This repository controls:

- public organization language
- terminology
- positioning
- documentation style
- agent instructions
- review standards
- public/private boundary rules
- AI assistance and agent boundary policy

The public profile README and organization-level documentation should align with the doctrine in `docs/`.

## Source of Truth

GitHub is the canonical source of truth for uScient public documentation.

External AI tools may review, summarize, or critique these documents, but their output does not become truth until it is reviewed, committed, and merged here.

## Agent Policy

Agents working in this repository must follow [`AGENTS.md`](AGENTS.md).

The standard is simple:

> Every sentence should make uScient more precise, more trustworthy, and more distinct.
